# 💼 Assignment 2: Personal Portfolio Website

This assignment demonstrates the design and development of a **personal portfolio website** using **HTML and CSS**, focusing on layout, styling, and user interface design.

---

## 🎯 Objective

* Create a personal portfolio webpage
* Design a clean and modern UI
* Showcase skills, projects, and personal details
* Use CSS for styling and layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ✨ Features

* 👤 Personal introduction section
* 🎨 Clean and modern UI design
* 🧑‍💻 Skills section with styled elements
* 📂 Projects section with card layout
* 📞 Contact section with details
* ✨ Hover effects and smooth styling
* 📱 Responsive design (basic)

---

## 📋 Sections Implemented

### 🔹 Hero Section

* Displays name and title
* Includes call-to-action button

### 🔹 About Section

* Short introduction about the user

### 🔹 Skills Section

* Lists technical skills (HTML, CSS, JS, etc.)

### 🔹 Projects Section

* Displays projects using card layout
* Includes title and description

### 🔹 Contact Section

* Shows email, GitHub, LinkedIn details
