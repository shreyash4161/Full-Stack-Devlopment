# Assignment 5 - Node.js + Express Website

This project is a simple travel agency website built with Node.js and Express.

## Features

- Homepage with featured travel packages
- Dynamic destination detail pages
- Booking form that stores submissions in a JSON file
- Bookings page to view submitted trip requests

## Run the project

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Notes

- MongoDB is not required for this version because bookings are stored in `data/bookings.json`.
- This JSON file acts like a lightweight NoSQL-style data store for the assignment.
